# Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow

This folder contains my practical exercises and notebooks from Chapters 1 to 19 of the book *Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow (2nd Edition)* by Aurélien Géron.

---

## Chapter Summary

| Chapter | Topic | Description |
| :--- | :--- | :--- |
| **01** | Machine Learning Landscape | Introduction to ML concepts, system types, and key challenges. |
| **02** | End-to-End ML Project | Building a complete housing regression project. |
| **03** | Classification | Classification tasks and performance metrics (Accuracy, Precision/Recall, ROC). |
| **04** | Training Models | Linear/Polynomial Regression, Gradient Descent, and regularization. |
| **05** | Support Vector Machines (SVM) | Linear/nonlinear SVM classification and regression, kernel tricks. |
| **06** | Decision Trees | Training, visualizing, and regularizing decision trees. |
| **07** | Ensemble Learning | Voting Classifiers, Bagging, Pasting, Boosting, and Stacking. |
| **08** | Dimensionality Reduction | Mitigating the curse of dimensionality using PCA. |
| **09** | Unsupervised Learning | Clustering (K-Means, DBSCAN) and Gaussian Mixtures. |
| **10** | Intro to ANNs with Keras | Implementing Perceptron and Multi-Layer Perceptron (MLP). |
| **11** | Training Deep Neural Networks | Addressing vanishing gradients, optimizers, transfer learning. |
| **12** | Custom Models with TensorFlow | Custom loss functions, layers, models, and training loops. |
| **13** | Loading & Preprocessing Data | Data API, TFRecord format, and preprocessing pipelines. |
| **14** | Deep Computer Vision | Convolutional Neural Networks (CNNs) and modern CNN architectures. |
| **15** | Processing Sequences (RNN/CNN) | Recurrent Neural Networks (RNNs), LSTM cells, GRUs, and 1D CNNs. |
| **16** | NLP with RNNs and Attention | Encoder-Decoders, Attention mechanisms, and Transformers. |
| **17** | Autoencoders and GANs | Representation and generative learning using Autoencoders and GANs. |
| **18** | Reinforcement Learning | Policy Gradients and Deep Q-Networks (DQN). |
| **19** | Scaling & Deploying Models | TF Serving, TF Lite, and distributed training on GPUs/TPUs. |
