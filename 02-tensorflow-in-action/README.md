# TensorFlow in Action

This folder contains my practical exercises and notebooks from Chapters 1 to 11 of the book *TensorFlow in Action* by Thushan Ganegedara.

---

## Chapter Summary

| Chapter | Topic | Description |
| :--- | :--- | :--- |
| **01** | World of TensorFlow | Introduction to TensorFlow components, hardware CPU vs GPU, and framework use cases. |
| **02** | TensorFlow 2 | Basic building blocks (tf.Variable, tf.Tensor) and matrix computations. |
| **03** | Keras & Data Retrieval | Keras APIs (Sequential, Functional, Subclassing) and data loading (tf.data). |
| **04** | Intro to Deep Learning | Building simple architectures: Autoencoders, CNNs, and RNNs. |
| **05** | Transformers | Transformer core concepts, self-attention, and encoder-decoder architecture. |
| **06** | Image Classification | Image EDA, Keras ImageDataGenerator, and Inception classifier. |
| **07** | Improving CNNs | Overfitting mitigation (data augmentation, dropout), transfer learning, and Grad-CAM. |
| **08** | Image Segmentation | Building tf.data pipelines for segmentation and implementing DeepLabv3. |
| **09** | Sentiment Analysis | Text preprocessing, LSTM sequence classification, and word embeddings. |
| **10** | Language Modeling | Generating text with GRUs, perplexity evaluation, and beam search. |
| **11** | Sequence-to-Sequence | Building an English-to-German translator using encoder-decoder seq2seq. |
