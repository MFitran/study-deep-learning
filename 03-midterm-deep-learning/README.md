# Midterm Deep Learning

This folder contains my portfolio projects for the Deep Learning Midterm Exam. It covers three core machine learning tasks using Python and scikit-learn.

---

## Project Summary

| Task | Notebook | Description | Metric / Key Result |
| :--- | :--- | :--- | :--- |
| **Classification** | `classification_fraud/classification_transaction_data.ipynb` | Credit Card Fraud Detection (Imbalanced dataset) | ROC-AUC: 0.7603, Recall: 0.65 |
| **Regression** | `regression_song/regression_hit_song.ipynb` | Continuous Target Prediction using Ridge Regression and LightGBM | R-squared: 0.3485, RMSE: 8.8055 |
| **Clustering** | `clustering_customer/clustering_customer_behavior.ipynb` | Credit Card Customer Segmentation using K-Means and PCA | 4 Customer Segments identified |
