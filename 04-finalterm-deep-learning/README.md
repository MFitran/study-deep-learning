# Finalterm Deep Learning

This folder contains my portfolio projects for the Deep Learning Final Exam. It covers advanced NLP tasks, fine-tuning pre-trained language models, and parameter-efficient fine-tuning (PEFT).

---

## Project Structure

| Task | Target Model | Dataset | Description |
| :--- | :--- | :--- | :--- |
| **Task 1** | BERT | AGNews, GoEmotions, MNLI | Text classification, emotion detection, and Natural Language Inference (NLI) fine-tuning. |
| **Task 2** | T5 | SQuAD | Question answering and text generation fine-tuning. |
| **Task 3** | Phi-2 (PEFT/LoRA) | XSum | Parameter-efficient fine-tuning for abstractive text summarization. |

---

## File Contents

- `task1/`: Sub-folders for `agnews/`, `goemotions/`, and `mnli/`. Each contains a notebook and reports.
- `task2/`: Contains notebooks and reports for SQuAD question answering fine-tuning.
- `task3/`: Contains the `fine_tune_phi2.ipynb` notebook and training adapters.
